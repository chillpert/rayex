Detailed Description

A class to create an ImGui-based GUI.

This class acts like an interface for the client to create their own GUI. It is possible to create multiple GUI objects and re-assign them to the renderer at runtime.

class CustomGui : public GuiBase
{
private:
// Configure style and input of the GUI.
void configure() override { }
// Put the rendering commands in here.
void render() override { }
};
Renderer myRenderer;
myRenderer.init( );
// Put this line after the declaration of the rx::Renderer object.
auto myGui = std::make_shared<CustomGui>( );
myRenderer.setGui( myGui );
Warning
The GUI object must be declared after the renderer to ensure the correct order for destroying and freeing resources.
